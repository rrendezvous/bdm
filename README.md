# Birthday Cat Surprise 🐱🎉

A super cute, interactive React birthday card built for Ma! This project features colorful custom SVGs, bouncy animations, and three adorable party cats (a tilapia cat, a fat cow cat, and an orange cat) to celebrate her special day.

## ✨ Features
- **Custom Loading Screen:** A playful animated paw print loader with fading dots.
- **Interactive Start Button:** A bouncy, friendly "Open your surprise!" button.
- **Three Core Kitties:** 
  - The Orange Cat
  - The Fat Cow Cat
  - The Tilapia Cat
- **Festive Animations:** A massive continuous confetti shower, floating balloons, gentle tail wags, and a sleepy yarn cat.
- **Responsive & Accessible:** Works beautifully on mobile and desktop devices.
- **Reduced Motion Support:** Respects system accessibility settings for a calmer experience if requested by the device.

## 🛠️ Tech Stack
- **React** (via Vite)
- **CSS3** (Custom Animations & Keyframes)
- **Inline SVGs** (Hand-crafted scalable vector graphics for crisp visuals)

## 🚀 Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open `http://127.0.0.1:5173/` in your browser!
